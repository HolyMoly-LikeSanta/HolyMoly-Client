## 프로젝트 소개
#### HolyMoly!는 "고맙다", "미안하다" 같은 진심 어린 자신의 마음을 솔직하게 표현하기 어려운 현대인들에게, 크리스마스를 핑계로 자신의 진심을 전할 수 있는 소통의 기회를 열고자 만들어진 웹/앱 서비스입니다. 
#### 내 파티를 꾸며보고, 주변에 초대장을 공유하여 파티에 초대해보세요! 초대받은 사람은 나에게 익명의 편지를 남길 수 있답니다.
